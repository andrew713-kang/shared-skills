# 📣 Digital Marketing Skill

## _데이터 기반 퍼포먼스 마케팅 및 콘텐츠 전략 엔진_

## 📋 When to Apply (발동 조건)

- "마케팅 전략 세워줘", "광고 집행 계획" 등 요청
- SEO/SEM 최적화 전략 수립
- 콘텐츠 마케팅 캘린더 및 바이럴 전략
- 퍼포먼스 마케팅 (Meta Ads, Google Ads) 최적화
- 인플루언서 마케팅 및 파트너십 전략

## 🎯 Mission (핵심 목표)

**한정된 예산으로 최대 ROI를 달성**합니다. 모든 마케팅 활동에 측정 가능한 KPI를 부여하고, 데이터 기반 A/B 테스트로 지속적으로 최적화합니다.

## 🔗 Analysis Chain (분석 파이프라인)

### Step 1: 디지털 현황 진단 (Digital Audit)

- 현재 웹사이트/앱 트래픽 분석
- 기존 마케팅 채널별 성과 리뷰
- 경쟁사 디지털 마케팅 벤치마킹
- **검색**: Tavily로 경쟁사 광고 전략, 키워드 동향 수집

### Step 2: 채널 전략 수립

- **프레임워크**: [채널 믹스 가이드](frameworks/channel-mix.md)

| 채널       | 목적          | 핵심 KPI                   | 예산 비율 |
| :--------- | :------------ | :------------------------- | :-------- |
| 검색 (SEO) | 유기적 유입   | 키워드 순위, 유기적 트래픽 | [%]       |
| 검색 (SEM) | 즉시 유입     | CPC, CTR, 전환율           | [%]       |
| SNS (유기) | 브랜드 인지도 | 팔로워, 참여율, 도달       | [%]       |
| SNS (유료) | 타겟 전환     | ROAS, CPA, 전환 수         | [%]       |
| 콘텐츠     | 신뢰 구축     | 체류 시간, 이탈률, 공유 수 | [%]       |
| 이메일     | 리텐션        | 오픈률, 클릭률, 전환율     | [%]       |

### Step 3: SEO 전략 (검색 최적화)

- 키워드 리서치 (검색량 × 경쟁 강도 × 전환 의도)
- 온페이지 최적화 (Title, Meta, H1, Internal Link)
- 기술적 SEO (Core Web Vitals, 사이트맵, 구조화 데이터)
- 콘텐츠 Gap 분석 (경쟁사는 있지만 우리는 없는 콘텐츠)

### Step 4: 콘텐츠 캘린더 설계

- 주간/월간 콘텐츠 발행 계획
- 콘텐츠 유형 믹스 (블로그, 영상, 인포그래픽, 케이스스터디)
- 각 콘텐츠의 퍼널 단계 매핑 (인지→관심→검토→전환)

### Step 5: 광고 캠페인 설계

- 캠페인 구조 (Campaign → Ad Set → Ad Creative)
- 타겟팅 전략 (인구통계, 관심사, 행동, 유사 타겟)
- A/B 테스트 계획 (크리에이티브, 카피, CTA, 랜딩페이지)

### Step 6: 성과 대시보드 + 최적화 사이클

- 핵심 KPI 실시간 추적 체계
- 주간 성과 리뷰 → 예산 재배분 → 크리에이티브 교체 사이클

## ✅ Quality Gates

### 반드시 포함

- [ ] 모든 채널에 **정량적 KPI** (CAC, ROAS, CPA) 설정
- [ ] **A/B 테스트** 계획 최소 3건 포함
- [ ] 예산 대비 **예상 ROI** 산출
- [ ] **주간 최적화** 사이클 정의

### 금지 사항

- ❌ KPI 없는 "브랜드 인지도 향상" 목표 금지
- ❌ 단일 채널 올인 전략 금지 (리스크 분산 필수)
- ❌ 타겟 정의 없는 광고 집행 금지
