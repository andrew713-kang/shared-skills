# 정책자금 신청 체크리스트

## 📋 사전 준비 체크리스트

### 기본 서류

- [ ] 사업자등록증 사본
- [ ] 법인등기부등본 (법인인 경우)
- [ ] 최근 3개년 재무제표 (감사보고서 포함)
- [ ] 부가가치세 과세표준증명원
- [ ] 국세/지방세 완납증명서
- [ ] 4대보험 완납증명서

### 사업계획서 핵심 항목

- [ ] 기업 및 대표자 소개 (경력, 수상, 특허)
- [ ] 기술 혁신성 (기존 대비 차별 포인트)
- [ ] 시장 분석 (TAM/SAM/SOM, 경쟁사)
- [ ] 매출 전망 (3~5개년, 근거 포함)
- [ ] 자금 사용 계획 (항목별 상세)
- [ ] 고용 계획 (증가 인원, 직무)

### 추가 서류 (해당 시)

- [ ] 기술 관련 인증서 (벤처인증, 이노비즈 등)
- [ ] 특허/실용신안 등록증
- [ ] 시제품/프로토타입 사진 또는 영상
- [ ] 주요 거래처 계약서 사본

## 🗓️ 신청 프로세스 타임라인

| 단계           | 소요 기간 | 핵심 활동                     |
| :------------- | :-------- | :---------------------------- |
| 1. 공고 확인   | -         | 기업마당/기관 사이트 모니터링 |
| 2. 서류 준비   | 2~4주     | 재무제표, 사업계획서 작성     |
| 3. 온라인 접수 | 1~2일     | 기관별 온라인 시스템          |
| 4. 서류 심사   | 2~4주     | 자격 요건 및 서류 적격 심사   |
| 5. 대면 평가   | 1일       | 발표(15분) + Q&A(15분)        |
| 6. 결과 통보   | 1~2주     | 선정/탈락 통보                |
| 7. 약정 체결   | 1~2주     | 보증서/대출 약정              |
| 8. 자금 집행   | 즉시~1주  | 계좌 입금                     |

## ⚠️ 사후관리 주의사항

- **용도 외 사용 금지**: 신청 항목과 다른 용도로 사용 시 → 부정수급
- **고용 유지 의무**: 약정 기간 내 고용 감축 시 → 자금 회수 가능
- **정기 보고**: 분기/반기별 사업 진행 보고 의무
- **중복 수혜 금지**: 동일 목적으로 복수 기관 지원 불가
