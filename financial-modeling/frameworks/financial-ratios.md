# 핵심 재무비율 분석 가이드

## 5대 카테고리

### 1. 수익성 (Profitability)

| 비율       | 공식            | 기준                   |
| :--------- | :-------------- | :--------------------- |
| ROE        | 순이익/자기자본 | 15%+ 우수              |
| ROIC       | NOPAT/투하자본  | WACC 초과 시 가치 창출 |
| 영업이익률 | 영업이익/매출   | 산업 평균 비교         |

### 2. 안정성 (Stability)

| 비율         | 공식              | 기준           |
| :----------- | :---------------- | :------------- |
| 부채비율     | 부채/자기자본     | 200% 이하 건전 |
| 이자보상배율 | 영업이익/이자비용 | 3배+ 안전      |
| 유동비율     | 유동자산/유동부채 | 150%+ 건전     |

### 3. 성장성 (Growth)

| 비율           | 공식             | 기준                          |
| :------------- | :--------------- | :---------------------------- |
| 매출성장률     | (당기-전기)/전기 | 산업 성장률 비교              |
| 영업이익성장률 | (당기-전기)/전기 | 매출보다 높으면 레버리지 효과 |

### 4. 효율성 (Efficiency)

| 비율         | 공식              | 기준          |
| :----------- | :---------------- | :------------ |
| 총자산회전율 | 매출/총자산       | 높을수록 우수 |
| 재고회전율   | 매출원가/평균재고 | 높을수록 우수 |

### 5. 현금 (Cash Flow)

| 비율      | 공식                               | 기준          |
| :-------- | :--------------------------------- | :------------ |
| FCF Yield | FCF/기업가치                       | 5%+ 매력적    |
| CCC       | 재고일수+매출채권일수-매입채무일수 | 짧을수록 우수 |

## 분석 원칙

- **3개년 이상** 추이 분석 필수
- **산업 평균** 대비 상대적 포지션 판단
- **카테고리 간 교차 분석** 필수
