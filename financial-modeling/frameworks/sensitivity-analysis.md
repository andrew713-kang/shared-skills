# 민감도 분석 (Sensitivity Analysis) 가이드

## 개요

핵심 가정 변동 시 결과 변화를 시각화하는 분석.
비유: "기온 1도 오르면 아이스크림 판매가 얼마나 오를까?"를 체계적으로 계산.

## 3가지 유형

### 1. Tornado Chart (1차원)

변수 하나씩 ±변동 → 영향 순위 정렬 → 가장 민감한 변수 파악

### 2. Data Table (2차원) — DCF에서 가장 많이 사용

| NPV (억원) | g=1% | g=2% | g=3% |
| :--------- | :--- | :--- | :--- |
| WACC 8%    | [값] | [값] | [값] |
| WACC 9%    | [값] | [값] | [값] |
| WACC 10%   | [값] | [값] | [값] |

### 3. 시나리오 분석

| 시나리오 | 가정                  | NPV | IRR | 판정        |
| :------- | :-------------------- | :-- | :-- | :---------- |
| 🟢 Best  | 성장률 +5%, 원가 -10% |     |     | Go          |
| 🟡 Base  | 현재 추세 유지        |     |     | Conditional |
| 🔴 Worst | 성장률 -3%, 원가 +15% |     |     | No-Go       |

## 결과 해석 원칙

- Worst에서도 NPV > 0 → **견고한 프로젝트**
- 변수 ±10%에 결과 50%+ 변동 → **해당 변수 집중 관리**
- Base=Go, Worst=No-Go → **Conditional: 리스크 헤지 필요**
