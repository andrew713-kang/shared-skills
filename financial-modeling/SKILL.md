# 💰 Financial Modeling Skill

## _CFO급 재무 분석 및 가치 평가 엔진_

## 📋 When to Apply (발동 조건)

- "재무 분석해줘", "사업 수익성 검토해줘" 등 재무 관련 요청
- 투자 유치/IR 자료 준비
- 자금 조달 옵션 비교 (부채 vs 증자 vs 정책자금)
- M&A 대상 기업 가치 평가
- 신규 사업의 손익분기점(BEP) 분석

## 🎯 Mission (핵심 목표)

**정량적 근거에 기반한 재무적 의사결정 지원**. 모든 수치에 산출 근거를 명시하고, 낙관/기본/비관 시나리오를 제시하여 리스크를 시각화합니다.

## 🔗 Analysis Chain (분석 파이프라인)

### Step 1: 재무제표 3대 분석

- 손익계산서(P&L): 매출, 매출원가, 영업이익, 순이익 추이
- 재무상태표(B/S): 자산, 부채, 자본 구조 분석
- 현금흐름표(C/F): 영업/투자/재무 활동 현금흐름
- **검색**: Tavily로 최신 금리, 환율, 산업 평균 재무비율 조회

### Step 2: 핵심 재무비율 진단

- **프레임워크**: [재무비율 가이드](frameworks/financial-ratios.md)
- 수익성(ROE, ROIC), 안정성(부채비율), 성장성, 효율성, 현금성

### Step 3: DCF 가치평가 모델

- **프레임워크**: [DCF 가이드](frameworks/dcf-model.md)
- FCF 5개년 추정 → WACC 산출 → 터미널 가치 → 기업가치(EV) → 주주가치

### Step 4: 민감도 분석

- **프레임워크**: [민감도 분석 가이드](frameworks/sensitivity-analysis.md)
- WACC ±1~2%p × 성장률 ±2~5%p 2차원 Data Table

### Step 5: 자금조달 옵션 비교

| 옵션      | 장점              | 단점                | 적합 상황       |
| :-------- | :---------------- | :------------------ | :-------------- |
| 은행 대출 | 지분 희석 없음    | 이자·담보 부담      | 안정적 현금흐름 |
| 정책자금  | 저금리, 보증 지원 | 자격 요건, 사후관리 | 중소·스타트업   |
| 투자 유치 | 대규모 자금       | 지분 희석, 경영권   | 고성장 기업     |
| 회사채    | 유연한 조건       | 신용등급 필요       | 중견기업 이상   |

### Step 6: Financial Dashboard 산출

- 참조: [Dashboard 템플릿](templates/financial-dashboard.md)
- 시나리오별 NPV/IRR 비교 → 최종 Go/No-Go/Conditional 판정

## ✅ Quality Gates

### 반드시 포함

- [ ] 모든 수치에 **산출 근거와 가정(Assumption)** 명시
- [ ] 환율/금리는 **당일 기준** 실시간 데이터
- [ ] 시나리오별 **NPV/IRR/BEP** 정량 비교
- [ ] 산업 평균 대비 **상대적 포지션** 분석 포함

### 금지 사항

- ❌ 출처 없는 추정치 사용 금지
- ❌ 단일 시나리오(낙관만 또는 비관만) 제시 금지
- ❌ 비용/리스크를 의도적으로 과소 평가 금지
