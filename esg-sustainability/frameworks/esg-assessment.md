# ESG 현황 진단 (Materiality Assessment) 가이드

## 개요

기업에 **가장 중요한(Material) ESG 이슈**를 식별하는 프로세스.
비유: 건강검진에서 **모든 항목을 검사**하되, 위험 수치가 나온 항목에 집중하는 것.

## 이중 중요성 (Double Materiality)

2025년 이후 글로벌 표준은 **이중 중요성** 평가를 요구합니다:

- **재무적 중요성**: ESG 이슈가 기업 재무에 미치는 영향
- **영향 중요성**: 기업 활동이 환경/사회에 미치는 영향

## E (환경) 진단 항목

| 항목        | 점검 내용                      | 데이터 소스              |
| :---------- | :----------------------------- | :----------------------- |
| 탄소 배출   | Scope 1/2/3 배출량             | 에너지 사용량, 연료 구매 |
| 에너지 효율 | 에너지 집약도, 재생에너지 비율 | 전기/가스 사용 데이터    |
| 물 관리     | 용수 사용량, 폐수 처리         | 수도 사용 데이터         |
| 폐기물      | 발생량, 재활용률               | 폐기물 처리 대장         |
| 생물 다양성 | 토지 사용, 생태계 영향         | 사업장 위치 분석         |

## S (사회) 진단 항목

| 항목      | 점검 내용                     | 데이터 소스      |
| :-------- | :---------------------------- | :--------------- |
| 노동 관행 | 임금, 근무시간, 산업안전      | 인사 데이터      |
| 인권      | 차별 금지, 강제노동 금지      | 고충 처리 시스템 |
| 다양성    | 성별/연령 다양성, 장애인 고용 | 인사 통계        |
| 공급망    | 협력사 ESG 관리               | 공급망 평가 결과 |
| 지역사회  | 사회 공헌, 지역 고용          | CSR 활동 보고    |

## G (거버넌스) 진단 항목

| 항목      | 점검 내용                  | 데이터 소스         |
| :-------- | :------------------------- | :------------------ |
| 이사회    | 독립성, 다양성, ESG 역량   | 이사회 구성 현황    |
| 윤리경영  | 반부패, 내부고발, 이해충돌 | 윤리강령, 제보 건수 |
| 정보공시  | 투명성, 적시성             | 공시 이력           |
| 세무 전략 | 조세 투명성                | 세무 보고           |
