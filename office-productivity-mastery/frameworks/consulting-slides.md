# 컨설팅 장표 표준 (Consulting Slide Standards)

## 개요

메시지가 한눈에 들어오는 **가독성 높은 슬라이드**를 만드는 규칙입니다.
"예쁜" 디자인보다 **"읽히는"** 디자인을 지향합니다.

## 1. 레이아웃 원칙 (Layout Principles)

### Lead Message (헤드라인)

- **위치**: 슬라이드 최상단 (전체 높이의 10~15%)
- **내용**: 해당 장표에서 하고 싶은 말 **단 한 문장** (두 줄 넘지 않기)
- **폰트**: 가장 크고 굵게 (본문의 1.5배 이상)

### Body (본문)

- **구조**: 좌우 2단(Comparison), 3단(Process), 혹은 4분면(Matrix) 구조 사용
- **텍스트**: 문장이 아닌 **개조식(Bullet points)** 사용
- **여백**: 콘텐츠 간 충분한 간격(White space) 확보

### Footer (바닥글)

- **좌측**: 출처(Source) - 데이터 신뢰도 확보
- **우측**: 페이지 번호(Page number) - 1/20 형식
- **중앙**: 대외비(Confidential) 표시

## 2. 시각화 규칙 (Visualization Rules)

### 차트 선택 가이드

- **비중(Share)**: 파이 차트 (항목 5개 이하)
- **추이(Trend)**: 라인 차트 (시간 축)
- **비교(Comparison)**: 바 차트 (수평/수직)
- **상관관계(Correlation)**: 산점도 (Scatter Plot)

### 컬러 사용

- **Main Color**: 브랜드 컬러 1가지 (강조용)
- **Sub Color**: 무채색 계열 (회색, 검정)
- **Signal**: 빨강(위험/하락), 파랑/초록(긍정/상승) 외에는 원색 자제

## 3. 폰트 가이드

- **국문**: 맑은 고딕, 나눔스퀘어, Pretendard (가독성 우선)
- **영문**: Arial, Roboto, Segoe UI
- **크기**:
  - 헤드라인: 20~24pt
  - 소제목: 16~18pt
  - 본문: 12~14pt (10pt 미만 금지)
