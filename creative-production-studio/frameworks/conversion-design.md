# 전환 유도 디자인 (Conversion-Focused Design)

## 개요

"예쁜" 디자인이 아니라 **"클릭하게 만드는"** 디자인 원칙입니다.
SNS 카드뉴스, 배너 광고, 랜딩 페이지에서 사용자의 행동(Action)을 유도합니다.

## 1. 시선 유도 패턴 (Eye Tracking Patterns)

### F-Pattern (텍스트 위주)

- 사용자는 상단 가로 → 중간 가로 → 좌측 세로 순으로 읽음.
- **적용**: 헤드라인을 상단에, 핵심 혜택을 중간 총알 아이콘(Bullet points)으로 배치.

### Z-Pattern (이미지/레이아웃 위주)

- 좌상단(로고) → 우상단(메뉴) → 좌하단(정보) → 우하단(CTA)
- **적용**: 시선의 종착지인 **우측 하단에 CTA 버튼("구매하기") 배치**.

## 2. 타이포그래피 계층 (Hierarchy)

| 계층         | 역할                 | 크기/굵기        | 컬러                 |
| :----------- | :------------------- | :--------------- | :------------------- |
| **Headline** | 시선 강탈, 핵심 후킹 | 가장 큼 / Black  | 강조색 (Brand Color) |
| **Sub-head** | 부연 설명, 신뢰도    | 중간 / Bold      | 짙은 회색            |
| **Body**     | 세부 정보            | 작음 / Regular   | 보통 회색            |
| **Caption**  | 유의사항, 출처       | 매우 작음 / Thin | 옅은 회색            |

## 3. CTA (Call To Action) 버튼 디자인

- **모양**: 둥근 모서리(Friendly) vs 직각(Formal)
- **컬러**: 배경색과 대비되는 보색(Complementary Color) 사용
- **문구**: 동사형으로 작성 ("제출" ❌ → "지금 무료로 받기" ⭕)
- **여백**: 버튼 주변에 여백(White space)을 두어 고립 효과(Isolation Effect) 활용

## 4. 고해상도 이미지 법칙

- 얼굴이 있는 이미지가 없는 이미지보다 주목도가 30% 높음 (Face Bias).
- 사람이 시선을 주는 방향(Gaze Cueing)에 중요한 텍스트를 배치하라.
