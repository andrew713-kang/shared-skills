# L10n(현지화) 체크리스트 (Localization Checklist)

## 개요

단순 번역(Translation)은 언어를 바꾸는 것이지만, **현지화(Localization)**는 제품을 그 나라의 문화와 환경에 맞게 재창조하는 것입니다.
"번역기 돌린 느낌"을 없애고 **Native-like** 경험을 제공하십시오.

## 1. 언어 및 텍스트 (Language)

- [ ] 문법 및 맞춤법 검수 완료 (Native Speaker Review)
- [ ] 존댓말/반말 체계 확인 (예: 한국어의 높임말, 독일어의 Sie/Du)
- [ ] 텍스트 길이 변화 대응 (독일어/러시아어는 영어보다 30% 길어짐)
- [ ] 날짜/시간 표기법 (MM/DD/YYYY vs DD/MM/YYYY)

## 2. 문화 및 상징 (Culture)

- [ ] 색상 의미 확인 (예: 중국에서 빨강은 행운, 서양에서 빨강은 위험/경고)
- [ ] 숫자 금기 (예: 서양의 13, 동양의 4)
- [ ] 제스처 및 아이콘 (예: 엄지척이 모욕인 국가 확인)
- [ ] 이미지 모델의 인종 다양성 확보

## 3. 기술 및 포맷 (Technical)

- [ ] 화폐 단위 및 기호 위치 ($10 vs 10€)
- [ ] 측정 단위 변환 (Inch/Feet ↔ cm/m, lb ↔ kg)
- [ ] 주소 입력 필드 (국가별 주소 체계 다름)
- [ ] 전화번호 입력 필드 (국가 번호 및 자릿수)

## 4. UI/UX

- [ ] **RTL(Right-to-Left)** 지원: 아랍어, 히브리어 사용 시 레이아웃 좌우 반전
- [ ] 폰트 가독성 및 특수문자 깨짐 확인 (CJK 문자셋)
