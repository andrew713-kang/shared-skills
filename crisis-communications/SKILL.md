# 🚨 Crisis Communications Skill

## _골든타임 48시간 위기 대응 및 평판 복구 엔진_

## 📋 When to Apply (발동 조건)

- 제품/서비스 리콜, 안전 사고 발생
- 부정적 언론 보도 또는 SNS 바이럴
- 임직원 비위, 법적 분쟁 공개
- 데이터 유출, 보안 침해 사고 (CISO와 협업)
- 고객 불만 폭증, 불매운동 징후

## 🎯 Mission (핵심 목표)

**위기를 기회로 전환**합니다. 골든타임(48시간) 내 신속하고 진정성 있는 대응으로 신뢰를 복구하고, 장기적 브랜드 레질리언스를 강화합니다.

## 🔗 Analysis Chain (분석 파이프라인)

### Step 1: 위기 상황 평가 (Situation Assessment)

- **프레임워크**: [위기 등급 분류 가이드](frameworks/crisis-classification.md)
- 위기 유형: 자연재해 / 운영 실패 / 인적 비위 / 외부 공격
- 심각도 등급: Level 1(경미) → Level 5(존립 위기)
- **검색**: Tavily로 실시간 언론/SNS 모니터링

### Step 2: 이해관계자 매핑

- 1차: 직접 피해자 (고객, 직원)
- 2차: 규제 기관, 파트너사, 투자자
- 3차: 미디어, 일반 대중, 커뮤니티
- 각 이해관계자별 핵심 메시지 차별화

### Step 3: 핵심 메시지 개발 (Message Triangle)

| 요소                  | 내용                                      |
| :-------------------- | :---------------------------------------- |
| **공감 (Empathy)**    | 피해자 입장을 인정하고 진심으로 유감 표명 |
| **행동 (Action)**     | 현재 취하고 있는 구체적 조치 설명         |
| **약속 (Commitment)** | 재발 방지를 위한 구체적 개선 계획         |

### Step 4: 채널별 대응 전략

| 채널           | 대응 시간  | 톤앤매너          | 핵심 주의사항         |
| :------------- | :--------- | :---------------- | :-------------------- |
| 공식 보도자료  | 24시간 내  | 공식적, 사실 기반 | 법적 리스크 검토 필수 |
| 기자 회견      | 48시간 내  | 진정성, 책임감    | 대표 직접 출석 권장   |
| SNS (즉시)     | 1~4시간 내 | 공감적, 투명      | 감정적 대응 금지      |
| 고객 직접 소통 | 즉시       | 개인화, 해결 중심 | 보상 기준 사전 합의   |

### Step 5: Q&A 시나리오 (예상 질문 대비)

- 최악의 질문 20개 리스트 + 승인된 답변
- "No Comment"를 피하는 브릿지 기법
- 법적 리스크가 있는 질문 대응 가이드

### Step 6: 복구 및 평판 재건 로드맵

- 즉시(24h): 공식 입장 + 사과 + 조치 공지
- 단기(1~4주): 개선 조치 실행 + 중간 보고
- 중기(1~3개월): 재발 방지 시스템 구축 + 외부 검증
- 장기(6~12개월): 브랜드 리포지셔닝 + 신뢰 회복 캠페인

## ✅ Quality Gates

### 반드시 포함

- [ ] 위기 등급(Level 1~5) **명확히 분류**
- [ ] 이해관계자별 **차별화된 메시지** 준비
- [ ] 최소 **20개 예상 질문 + 승인 답변** 목록
- [ ] **법적 리스크 검토** 완료 표시

### 금지 사항

- ❌ 책임 회피 또는 타인 전가 권고 금지
- ❌ "No Comment" 기본 전략 금지
- ❌ 감정적/공격적 톤 사용 금지
- ❌ 비공식 채널(개인 SNS)로 대응 권고 금지
