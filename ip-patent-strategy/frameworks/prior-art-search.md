# 선행기술 조사 (Prior Art Search) 가이드

## 개요

특허 출원 전 유사 기술의 존재를 확인하여 **특허 가능성**을 사전 판단합니다.
비유: 부동산 계약 전에 **등기부등본을 확인**하는 것과 같습니다.

## 주요 검색 DB

| DB                  | 관할   | URL                     | 검색 키워드        |
| :------------------ | :----- | :---------------------- | :----------------- |
| **KIPRIS**          | 한국   | kipris.or.kr            | 한국 특허/실용신안 |
| **USPTO**           | 미국   | patents.google.com      | 미국 특허/출원     |
| **EPO (Espacenet)** | 유럽   | worldwide.espacenet.com | 유럽+글로벌        |
| **WIPO**            | 국제   | patentscope.wipo.int    | PCT 출원           |
| **Google Patents**  | 글로벌 | patents.google.com      | 통합 검색          |

## 검색 전략

### 1. 키워드 검색

- 핵심 기술 용어 + 유사어/동의어 조합
- 한글 + 영어 병행 검색

### 2. IPC/CPC 분류코드 검색

- 기술 분야별 국제특허분류(IPC) 코드 활용
- 더 정밀한 기술 분야 특정 가능

### 3. 인용/피인용 분석 (Citation Tree)

- 핵심 선행문헌의 인용 관계를 추적
- 기술 발전 흐름 파악

## 결과물 형식

| 순번 | 선행문헌 번호 | 출원인   | 핵심 청구항 | 유사도         | 대응 전략            |
| :--- | :------------ | :------- | :---------- | :------------- | :------------------- |
| 1    | [번호]        | [출원인] | [내용]      | 높음/중간/낮음 | 회피설계/차별화/포기 |

## 주의사항

- 최소 **3개 DB** 교차 검색 (한국+미국+글로벌)
- **비특허 문헌**(논문, 기술 블로그)도 선행기술에 해당
- 조사 결과가 출원 가능성을 **보장하지 않음** (최종 판단은 변리사)
