# IP 랜드스케이프 분석 가이드

## 개요

특정 기술 분야의 **특허 지형도**를 그려 경쟁 현황과 기회 영역을 파악합니다.
비유: 전쟁 전 **위성 사진으로 적군의 진지 배치**를 파악하는 것.

## 분석 절차

### 1. 기술 분야 정의

- 분석 대상 기술의 범위(Scope) 설정
- IPC/CPC 코드 기반으로 기술 영역 구획

### 2. 출원 동향 분석 (Macro View)

- 연도별 출원 건수 추이 → 기술 성숙도 판단
- 급증기(Growth) / 정체기(Maturity) / 감소기(Decline) 구분

### 3. 핵심 출원인 분석 (Player Map)

| 출원인     | 출원 건수 | 핵심 기술 | r&d 투자 추정 | 위협 수준 |
| :--------- | :-------- | :-------- | :------------ | :-------- |
| [경쟁사 A] | [건]      | [기술]    | [규모]        | H/M/L     |

### 4. 기술-시장 매트릭스

|                    | 시장 성장 High                     | 시장 성장 Low     |
| :----------------- | :--------------------------------- | :---------------- |
| **특허 밀집 High** | 🔴 Patent Thicket (회피 설계 필요) | ⚠️ 과잉 투자 영역 |
| **특허 밀집 Low**  | 🟢 기회 영역 (선점 가능)           | 💤 관심 낮은 영역 |

### 5. 빈틈 분석 (White Space)

- 특허가 적지만 시장 가치가 높은 영역 식별
- 해당 영역의 출원 전략 수립

## 결과물

- 기술별 출원 동향 차트
- 핵심 출원인 비교 매트릭스
- 기술-시장 매트릭스 (2×2)
- White Space 목록 + 선점 전략 권고
