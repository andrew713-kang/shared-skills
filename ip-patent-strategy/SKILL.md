# 🛡️ IP & Patent Strategy Skill

## _글로벌 지식재산 전략 및 기술 보호 엔진_

## 📋 When to Apply (발동 조건)

- "특허 출원 가능해?", "기술 보호 전략 세워줘" 등 IP 관련 요청
- 신기술/신제품 출시 전 IP 리스크 사전 점검
- 경쟁사 특허 침해 여부 분석 (Freedom-to-Operate)
- 특허 포트폴리오 가치 평가 및 라이선싱 전략
- M&A 시 IP 실사 (Due Diligence)

## 🎯 Mission (핵심 목표)

**핵심 기술을 체계적으로 보호**하고, IP를 **수익화 자산**으로 전환하는 전략을 수립합니다. 방어적 보호와 공격적 활용의 균형을 잡습니다.

## 🔗 Analysis Chain (분석 파이프라인)

### Step 1: IP 자산 현황 진단

- 보유 특허/출원 중 특허 목록화
- 상표권, 디자인권, 영업비밀 현황
- IP 성숙도 평가 (초기/성장/성숙/방어)

### Step 2: 선행기술 조사 (Prior Art Search)

- **프레임워크**: [선행기술 조사 가이드](frameworks/prior-art-search.md)
- **검색**: Exa로 관련 특허 DB(KIPRIS, USPTO, EPO) 검색
- 핵심 기술의 특허 가능성(Patentability) 사전 판단

### Step 3: 경쟁사 IP 랜드스케이프 분석

- **프레임워크**: [IP 랜드스케이프 가이드](frameworks/ip-landscape.md)
- 경쟁사 특허 출원 동향 매핑 (기술 분야 × 시기)
- 특허 벽(Patent Thicket) 및 회피 설계 필요 영역 식별
- **검색**: Tavily로 최신 특허 분쟁 사례 및 판결 동향

### Step 4: IP 전략 옵션 도출

| 전략            | 용도                         | 리스크             |
| :-------------- | :--------------------------- | :----------------- |
| 직접 출원       | 핵심 기술 독점 보호          | 비용, 공개 의무    |
| 영업비밀 유지   | 리버스엔지니어링 어려운 기술 | 유출 리스크        |
| 라이선싱        | IP 수익화, 시장 확대         | 경쟁자 강화 가능성 |
| 크로스 라이선싱 | 분쟁 해결, 기술 접근         | 협상력 필요        |
| 방어적 출원     | 경쟁사 견제, FTO 확보        | 유지 비용          |

### Step 5: IP 로드맵 + 비용 추정

- 출원 국가별 전략 (한국/미국/유럽/중국/일본)
- 3개년 IP 포트폴리오 확장 계획
- 예상 비용 (출원비, 심사비, 유지비, 대리인 비용)

## 🔍 AI Search Integration

| 검색 엔진  | 검색 대상                                           | 활용 단계 |
| :--------- | :-------------------------------------------------- | :-------- |
| **Exa**    | KIPRIS/USPTO 특허 검색, 학술 논문, 기술 동향        | Step 2, 3 |
| **Tavily** | 최신 특허 분쟁 판결, IP 정책 변화, 산업별 IP 트렌드 | Step 3, 4 |

## ✅ Quality Gates

### 반드시 포함

- [ ] 선행기술 조사 최소 **3개 DB** 교차 검색
- [ ] 경쟁사 특허 포트폴리오 **시각적 매핑** (기술 × 시기)
- [ ] 국가별 출원 전략 및 **비용 추정** 포함
- [ ] FTO(자유실시) 리스크 평가 포함

### 금지 사항

- ❌ 특허 가능성을 보장하는 표현 금지 (법적 판단은 변리사 영역)
- ❌ 선행기술 조사 없이 출원 추천 금지
- ❌ 단일 국가 전략만 제시 금지 (글로벌 고려 필수)
