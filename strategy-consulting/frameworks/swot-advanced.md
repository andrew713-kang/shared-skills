# SWOT Advanced 가이드 (가중치 정량 분석)

## 개요

단순 4칸 SWOT을 넘어 **가중치 기반 정량적 전략 우선순위**를 도출합니다.
비유: "우리가 잘하는 것"을 나열하는 대신, **"얼마나 잘하는지 점수를 매겨"** 비교.

## Phase 1: 요소 나열 (5~8개씩)

- **S** (강점): 내부 역량 중 경쟁우위
- **W** (약점): 개선이 필요한 내부 요소
- **O** (기회): PESTEL/5Forces에서 도출된 외부 기회
- **T** (위협): PESTEL/5Forces에서 도출된 외부 위협

## Phase 2: 가중치 부여

| 강점    | 중요도(1~5) | 현재 수준(1~5) | 가중 점수 |
| :------ | :---------- | :------------- | :-------- |
| [강점1] | 5           | 4              | 20        |

## Phase 3: TOWS 교차 전략 도출

|                   | Strengths             | Weaknesses            |
| :---------------- | :-------------------- | :-------------------- |
| **Opportunities** | **SO** 공격적 성장    | **WO** 약점 보완 성장 |
| **Threats**       | **ST** 강점 활용 방어 | **WT** 리스크 최소화  |

## 결과물

- SWOT 가중치 매트릭스 (4개 표)
- TOWS 교차 전략 (최소 4개 전략 옵션)
- 전략 우선순위 랭킹 (가중 점수 기반)
