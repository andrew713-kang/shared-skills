# BCG Growth-Share Matrix 가이드

## 개요

BCG의 사업 포트폴리오 분석 도구.
비유: 여러 사업을 **⭐Star, 🐄Cash Cow, ❓Question Mark, 🐕Dog**로 분류하여 자원 배분 결정.

## 4분면

| 분면             | 시장성장률 | 시장점유율 | 전략              |
| :--------------- | :--------- | :--------- | :---------------- |
| ⭐ Star          | 높음       | 높음       | 집중 투자         |
| 🐄 Cash Cow      | 낮음       | 높음       | 현금 수확         |
| ❓ Question Mark | 높음       | 낮음       | 선별 투자 or 철수 |
| 🐕 Dog           | 낮음       | 낮음       | 매각/구조조정     |

## 분석 절차

1. 사업 단위 정의
2. 시장 성장률(CAGR) 산정
3. 상대적 점유율 측정 (vs 최대 경쟁사)
4. 2×2 매트릭스 배치
5. 분면별 자원 배분 전략 결정

## 함께 사용하면 좋은 도구

- **Ansoff Matrix**: 성장 방향 구체화
- **Blue Ocean Canvas**: 새로운 시장 공간 탐색
- **GE-McKinsey Matrix**: 더 세밀한 9칸 분석

## 주의사항

- Dog이라도 **방어적 가치, 시너지 효과** 검토 필수
- 시장 데이터는 반드시 **실시간 검색**으로 최신 수치 확인
